Projects

Below are some Data Analytics projects on SQL, Python, Tableau and ML.

Maximizing Zara’s Gross Margin by Data-Driven Retail Strategy

The project focused on optimizing Zara’s Gross Margin through data-driven insights by leveraging 6.4 million rows of global retail transactions dataset. The analysis includes data cleaning, feature engineering, and building visualization to enhance business decision-making.

Analysing Amazon Brazil’s Dataset for Indian Market Strategy

The project was to analyze Amazon Brazil's retail data to identify trends and customer behaviors that can be leveraged in the Indian market. The insights that were delivered were to help Amazon India make data-driven decisions, enhance customer experience, and seize new opportunities.

Detailed Explaratory Analysis of Bumble User Profile using Python

This project was to conduct an exploratory analysis of Bumble user profiles dataset to uncover insights into demographics, preferences, and behavioral trends. The findings were aimed at to help improve user engagement/usage, matchmaking algorithms and platform personalization.




Health Insurance Data Regression Analysis using Excel

Validated key prerequisites and conditions essential for accurate regression analysis using Excel charts, pivot tables, and ANOVA tests. Evaluated relationships through detailed charting and aggregations. Used Excel's ANOVA feature to assess statistical significance and variability across groups.

Flipkart, one of India’s leading e-commerce platforms was facing a decline in customer retention. This project analyzes customer service call center data to uncover insights into customer service performance and its impact on customer satisfaction and retention. Used Aggregations, Excel Charts.

Swiggy Web Traffic and User Engagement Dashboard

Swiggy was experiencing high web traffic but lower-than-expected conversion rates. This project aimed to analyze user behavior and traffic patterns to uncover actionable insights that can help Swiggy’s marketing team optimize digital strategy, enhance user engagement, improve conversion rate.
